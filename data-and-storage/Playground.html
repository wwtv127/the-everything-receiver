R1 SDK Interface Detector

This tool tests for custom events exposed by the RabbitOS Creations SDK.

How to Use:

  • Load this page on your Rabbit R1.
  • **Press PTT or Action:** Look for changes in the **"Custom Events (PTT/Action)"** area.
  • **Move Scroll Wheel:** Look for changes in the **"Scroll Wheel Debugger"** area.
  • If an API exists, it will reveal the correct event type or method name!
